# eslint-config-reearth

An extensible shared ESLint config for Re:Earth projects

## How to install

```
npm install eslint prettier eslint-config-reearth --save-dev
```

Then edit `.eslintrc.js`:

```
module.exports = {
  extends: ["reearth"]
};
```
